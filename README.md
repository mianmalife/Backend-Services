# Express API Services

## 🔨 技术栈

**Server:** Node.js, Express, Session, MongoDB

## 💼 API

**用户注册**
```
POST /api/users/register
```
**用户登录**

```
POST /api/users/login
```
**用户登出**

```
GET /api/users/logout
```
**获取用户信息**

```
GET /api/users/profile
```

## 📦 Install

```bash
npm install
```

## 💡 Run

```bash
npm run start
```
